# Biblioteca App

Aplicação web para gerenciamento de livros feita com JavaScript puro.

## Demo

Acesse: (https://biblioteca-liard-mu.vercel.app/)

## Funcionalidades

- Adicionar e remover livros
- Marcar como lido ou não lido
- Busca em tempo real
- Ordenação por título, páginas e status
- Persistência com localStorage

## Conceitos

- Programação orientada a objetos (constructors e prototype)
- Manipulação de DOM
- Gerenciamento de estado
- Event delegation
- Separação entre dados e interface

## Tecnologias

- JavaScript
- HTML
- CSS
